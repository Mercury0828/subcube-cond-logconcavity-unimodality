`D_yes`: the uniform product distribution `U_n`. This is SR.

`D_no`: choose a hidden unordered pair `e={a,b}` uniformly from `binom(n,2)`. Define

\[
\mu_e(s)=2^{-n}(1+\rho s_a s_b),
\]

with `0 < ρ ≤ 1/2`. Equivalently, on the hidden pair,

\[
P(00)=P(11)=\frac{1+\rho}{4},\qquad
P(01)=P(10)=\frac{1-\rho}{4},
\]

and all other coordinates are independent fair bits.

For every fixed `e`,

\[
\operatorname{Cov}_{\mu_e}(X_a,X_b)=\frac{\rho}{4}>0.
\]

Since SR implies negative association, hence all pairwise covariances are `≤0`, Lemma C0-L1 gives

\[
\operatorname{dist}_1(\mu_e,\mathrm{SR}) \ge \frac{1}{3}\cdot \frac{\rho}{4}=\frac{\rho}{12}.
\]

Set `ρ = 12ε` for small enough `ε`, say `ε ≤ 1/24`. Then every `D_no` instance is **ε-far from SR in ℓ₁**, certified by the **pairwise-positive-correlation Lipschitz margin**.

**Adaptive SUBCOND Indistinguishability**

For a subcube query with free set `R` and fixed set `F`, under hidden edge `e={a,b}`:

- if `e ⊆ F`, the conditional sample over `R` is uniform;
- if exactly one endpoint is free, the free endpoint has a `±ρ` bias;
- if both endpoints are free, the two free endpoints have correlation `ρ`.

Thus each query exposes at most one character of the hidden edge: a singleton character or a pair character.

For a deterministic adaptive tester, compare transcript laws under `U_n` and the mixture over hidden `e`. Under the null transcript measure, each query induces a partition of the `N=binom(n,2)` possible hidden edges into character classes. Every non-null class has size at most `n`: crossing edges sharing the same free endpoint are the largest classes.

A standard adaptive collision/χ² argument gives

\[
\chi^2(\mathsf{Trans}_{D_no},\mathsf{Trans}_{D_yes})
\le
\frac{2}{n-1}\bigl((1+\rho^2)^q-1\bigr).
\]

Sketch of the proof: for two independent hidden edges `e,e'`, the likelihood-ratio second moment only grows on queries where `e` and `e'` induce the same non-null character. For any nonempty set of query times, conditioned on the null transcript, the probability of simultaneous collision is at most `n / binom(n,2) ≤ 2/(n-1)`. Expanding the product yields the displayed bound. This handles adaptivity because the query partitions are fixed after conditioning on the null transcript.

Therefore, if

\[
q \le c \frac{\log n}{\rho^2},
\]

the transcript total variation is `<1/3`. With `ρ=Θ(ε)`, this gives

\[
q = \Omega\!\left(\frac{\log n}{\epsilon^2}\right).
\]

**Exponent / Tightness**

The certified exponent is only

\[
\widetilde{\Omega}(n^0/\epsilon^2)
\]

with a `log n` factor. This **does not match** any plausible `\widetilde O(\sqrt n/\epsilon^2)` or `\widetilde O(n/\epsilon^2)` C1-style upper bound. It leaves a polynomial gap and must not be called tight.
